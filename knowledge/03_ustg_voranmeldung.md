# UStG §18 Abs. 2, 2a — Advance VAT Returns (Umsatzsteuer-Voranmeldung)

**Source:** Umsatzsteuergesetz (UStG), Section 18(2) and (2a)
**URL:** https://www.gesetze-im-internet.de/ustg_1980/__18.html

## Text (excerpt, translated)

The advance return period (Voranmeldungszeitraum) is the calendar quarter. It becomes the
calendar month where the tax owed for the preceding calendar year exceeded EUR 7,500. Where
the preceding year's tax did not exceed EUR 2,000, the tax office may release the
entrepreneur from the obligation to file advance returns entirely.

Entrepreneurs newly commencing activity must file monthly advance returns for the current
and following calendar year, regardless of the amounts involved (Neugründerregelung), for
periods through the end of 2027 under current transitional rules.

Advance returns must be filed electronically, using the officially prescribed data set, by
the 10th day following the end of the advance return period. A permanent extension
(Dauerfristverlängerung) of one month is available on application, generally conditioned on
an advance special payment (Sondervorauszahlung) of one-eleventh of the prior year's
advance-payment total.

## Practical note

Newly founded businesses should plan for monthly filing regardless of turnover size in their
first two years, and should apply for Dauerfristverlängerung early if a one-month buffer on
deadlines is needed.
